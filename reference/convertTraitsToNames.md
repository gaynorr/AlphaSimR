# Convert traits to a vector of names

This function processes the traits arguments from GS models. It usually
receives a number indicating the trait, but may also be the trait name
itself or a custom function.

## Usage

``` r
convertTraitsToNames(traits, simParam)
```

## Arguments

- traits:

  the traits argument from a GS model

- simParam:

  the simulation parameters object

## Value

a vector of names for traits
