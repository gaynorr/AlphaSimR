# Calculate phenotypes

Calculate phenotypes

## Usage

``` r
calcPheno(pop, varE, reps, p, traits, simParam)
```

## Arguments

- pop:

  an object of class Pop

- varE:

  a vector or matrix of residual variances

- reps:

  number of reps for phenotype

- p:

  p-value for environment

- traits:

  number of traits

- simParam:

  simulation parameters object
