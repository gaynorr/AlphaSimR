# Sample additive effects

Samples deviates from a normal distribution or gamma distribution with a
random sign

## Usage

``` r
sampAddEff(qtlLoci, nTraits, corr, gamma, shape)
```

## Arguments

- qtlLoci:

  total number of loci

- nTraits:

  number of traits

- corr:

  correlation between traits

- gamma:

  indicator of trait should use a gamma distribution

- shape:

  gamma distribution shape parameter

## Value

a matrix with dimensions qtlLoci by nTraits
